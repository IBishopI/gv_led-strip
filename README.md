# Rainbow Trail - Lights ![](/docs/badge_rtl.svg)

![](/docs/logosmall.png)


This project makes possible to control animated address led strip via voice assistants.

## Tech stack:
- Voice assitant -> Webhook -> MQTT -> ESP32

## Available modes:
> Solid colors:
- White
- Red
- Tomato
- Magenta
- Hot pink
- Pink
- Coral
- Orange
- Yellow
- Lime
- Green
- Olive
- Aqua
- Teal
- Blue
- Indigo
- Navy
- Amethyst
- Violet
> Animated modes:
- Rainbow
- pencil
- Heat
- Lava
- Forest
- Clouds
- Ocean
- Party
- Skittles
- Twinkle
- Christmas
- Rave
- Pacman
